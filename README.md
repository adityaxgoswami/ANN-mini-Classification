# ANN Mini Classification

A mini project demonstrating how to build, train, and evaluate an Artificial Neural Network (ANN) for classification tasks using Deep Learning.

This project covers the complete pipeline from preprocessing data to training a neural network and evaluating its performance.

---

## Project Overview

This project focuses on:

- Data preprocessing
- Building an Artificial Neural Network (ANN)
- Training using TensorFlow / Keras
- Model evaluation
- Classification predictions

The goal is to understand the fundamentals of neural networks and implement a simple classification pipeline.

---

## Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook
